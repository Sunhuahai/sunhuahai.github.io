Linux命令总结#

  1. 命令格式：
    命令 + <选项> + 目标文件或目录

    • /：根目录
    • ./：当前目录 (执行命令如./a.out)
    • ~/：用户目录

  2. 目录操作：

    • ls：列出目录下所有文件
      • ls -al：显示详细信息和隐藏文件
    • cd <目录>：进入指定目录
      • cd ..：返回上级目录
      • cd ~：返回用户目录
    • pwd：显示当前目录路径
    • mkdir <目录名>：创建子目录
    • rmdir <目录名>：删除子目录

  3. 文件操作：

    • touch <文件名>：创建文件
    • cp <源文件> <目标文件>：复制文件
    • mv <源文件> <目标文件>：移动文件或重命名
    • rm <文件名>：删除文件
      • rm -rf <目录>：强制递归删除

  4. 文件显示：

    • cat <文件名>：显示文件内容
    • more <文件名>：分页显示文件
    • head/tail -n <行数> <文件名>：显示文件头部/尾部若干行

  5. 其他命令：

    • ln -s <源文件> <链接名>：创建符号链接
    • diff <文件1> <文件2>：比较文件
    • chmod +x <文件名>：更改文件权限
    • grep <模式> <文件>：文本搜索
    • find <路径> -name <文件名>：查找文件

  6. 压缩与解压：

    • tar cvf <包名>.tar <目录>：打包
    • tar xvf <包名>.tar：解包
    • gzip <文件>：压缩
    • gunzip <文件.gz>：解压

  7. 管道与重定向：

    • >：重定向输出到文件
      • ls -l > out.txt
    • |：管道符
      • ls | grep <模式>
    • >>：追加输出到文件

  8. 系统操作：

    • echo <变量>：显示变量值
    • ps：查看活跃进程
    • kill <进程ID>：终止进程
    • history：查看历史命令
    • du -h <目录>：查看目录占用空间

  9. 网络操作：

    • ssh <用户>@<主机>：登录远程主机
    • scp <源文件> <用户>@<主机>:<目录>：复制文件到远程
    • rsync：同步文件
